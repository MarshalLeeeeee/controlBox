# controlBox

## feature
 * Camera mode: first person, i.e. the camera is fixed at the tail of the box
 * Keyboard control: 
	 * w: move box forward
	 * a: turn left
	 * s: move box backward
	 * d: turn right
 * Mouse control: free control of the view direction
 * Collision: cannot move the box with penetration toward the wall
